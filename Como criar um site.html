Você disse:
como criar um site

O ChatGPT disse:

Criar um site hoje é bem mais simples do que parece 🙂 Vou te explicar do jeito prático, do básico ao completo. Você escolhe o nível que combina com você.

🔹 1. Defina o objetivo do site

Antes de qualquer coisa, pense:

  • É um site pessoal?

  • Empresa / portfólio?

  • Loja online?

  • Blog?

Isso define a ferramenta certa.

🔹 2. Escolha a forma de criar

👉 Opção 1: Sem programar (mais fácil)

Ideal se você quer rapidez.

Plataformas:

  • Wix

  • WordPress

  • Shopify (loja)

Vantagens:

  • Arrastar e soltar

  • Templates prontos

  • Não precisa saber código

👉 Opção 2: Programando (mais profissional)

Ideal se quer personalização total.

Você vai precisar aprender:

  • HTML → estrutura do site

  • CSS → design

  • JavaScript → interatividade

Ferramentas comuns:

  • Visual Studio Code (editor)

  • GitHub (hospedagem grátis)

🔹 3. Registrar um domínio (endereço do site)

Exemplo: www.seusite.com.br

Onde registrar:

  • Registro.br (.br)

  • HostGator

🔹 4. Hospedar o site

A hospedagem é onde o site fica armazenado.

Opções populares:

  • Hostinger

  • Locaweb

  • GitHub Pages (grátis)

🔹 5. Crie o conteúdo

  • Textos claros

  • Imagens leves

  • Página inicial

  • Sobre

  • Contato (WhatsApp, e-mail)

🔹 6. Publique e divulgue

  • Teste no celular

  • Conecte ao Google

  • Divulgue nas redes sociais

💡 Quer um atalho?

Se você me disser:

  • se é site pessoal, empresa ou loja

  • se quer sem programação ou com código

eu te passo um passo a passo exato, já no seu perfil 😉

function filtrar(){
  const categoria = document.getElementById("filtroCategoria").value;
  const preco = document.getElementById("filtroPreco").value;

  document.querySelectorAll(".product").forEach(produto=>{
    let mostrar = true;
    const valor = parseFloat(produto.dataset.preco);

    if(categoria !== "todos" && produto.dataset.categoria !== categoria){
      mostrar = false;
    }

    if(preco === "baixo" && valor > 100) mostrar = false;
    if(preco === "medio" && (valor < 100 || valor > 150)) mostrar = false;
    if(preco === "alto" && valor < 150) mostrar = false;

    produto.style.display = mostrar ? "block" : "none";
  });
}

let carrinho = JSON.parse(localStorage.getItem("carrinho")) || [];
let itensDiv = document.getElementById("itens");
let total = 0;

function renderCarrinho(){
  itensDiv.innerHTML = "";
  total = 0;

  if(carrinho.length === 0){
    itensDiv.innerHTML = "<p>Carrinho vazio.</p>";
    document.getElementById("subtotal").innerText = "";
    return;
  }

  carrinho.forEach((item, index)=>{
    total += item.preco;
    itensDiv.innerHTML += `
      <p>${item.nome} — R$ ${item.preco.toFixed(2)}
      <button onclick="remover(${index})">❌</button></p>
    `;
  });

  document.getElementById("subtotal").innerText =
    "Subtotal: R$ " + total.toFixed(2);
}

function remover(index){
  carrinho.splice(index,1);
  localStorage.setItem("carrinho", JSON.stringify(carrinho));
  renderCarrinho();
}

function limparCarrinho(){
  localStorage.removeItem("carrinho");
  carrinho = [];
  renderCarrinho();
}
